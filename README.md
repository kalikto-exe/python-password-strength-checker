# Password Strength Checker (Python)

A simple Python script that checks how strong a password is
based on length, character variety, and common security rules.

## What this project does
- Checks password length
- Detects uppercase, lowercase, numbers, and symbols
- Gives a strength rating (Weak / Medium / Strong)
- Explains why a password is weak

## Tech used
- Python 3

## How to run
```bash
python password_checker.py
